AdtToAS
=======

A simple shell program to migrate (or convert) your existing Android ADT projects to Android Studio Projects. 

Usage
-----
Clone the repo and run the `adt2as` in your terminal.
```bash
$ ./adt2as my_adt_project new_as_project
```
